Rede Neural Convolucional (convnet)¶

Um tipo de modelo de deep-learning muito utilizado para resolver problemas de classificao de imagens.

Diferena fundamental: Camadas densas aprendem padres globais do seu espao amostral enquanto as camadas convolutivas aprendem padres locais.

Esta caracterstica chave das convnets implica nessas duas propriedades interessantes:

  • Os padres que ela aprende so invariantes em relao a translao
  • Elas podem aprender hierarquias espaciais dos padres

As convolues operam em cima de tensores 3D, chamados feature maps, com dois eixos espaciais (width e height) e um de profundidade (depth ou channel).

Exemplo: para uma imagem RGB, a dimenso do eixo profundidade 3: uma para cor vermelha, outra para cor verde e outra para cor azul.

Feature Map: Cada dimenso do eixo profundidade uma feature (ou filtro).

Output Feature Map: Continua um tensor 3D: com altura e largura. Porem, a profundidade pode ser arbitrria, j que a profundidade de sada um parmetro da camada e os difentes canais no eixo profundidade no mais representam uma cor RGB, ao invs disso, eles representam filtros.

Filters: Os filtros codificam um aspcto especfico dos dados de entrada: em alto nvel, um nico filtro pode codificar o conceito "presena de um olho", por exemplo.

Explicação da convolução nos dados MNIST¶

No exemplo dos dgitos MNIST, a primeira camada de convoluo recebe uma feature map de tamanho (28, 28, 1) e gera uma feature map de tamanho (26, 26, 32). Ela calcula 32 filtros em cima da entrada. Cada um desses 32 canais de sada contem uma tabela de valores 26x26, que um mapa de respostas do filtro sobre a entrada, indicando a resposta do padro de um filtro em diferentes locais na entrada.

Operação Max Pooling¶

Objetivo: Reduzir a dimenso das feature maps.

Max pooling consiste em extrair janelas das feature maps de entrada e gerar o mximo valor de cada canal. conceitualmente silimar a convoluo, com excesso de que, ao invs de transformar caminhos locais usando transformao linear, usada uma operao tensorial "mximo".

while (i < (len(test_lbl) - 2) and len(idx) <= max_samples):

    idx.append(i)
  i += 1

figures_per_line = 4
lines = np.ceil(max_samples / figures_per_line)

plt.subplots_adjust(hspace=.5)
i = 0
while (i < len(idx)):
  j = idx[i]
  plt.subplot(lines, figures_per_line, i + 1)
  plt.imshow(test_img[j], cmap=plt.get_cmap('gray'))
  plt.title('Convnet: %s\nDense: %s\nTarget: %s' % 
            (conv_predict[j], dense_predict[j], test_lbl[j]),
            fontsize=14)
  plt.xticks([])
  plt.yticks([])
  i += 1
plt.show()

img_urls = [
  'https://i.imgur.com/JscD1yr.jpg',
  'https://i.imgur.com/4t0gdrO.jpg'
]

target_set = [
  7,
  2
]

data_set = []

for url in img_urls:
  response = requests.get(url)
  img = Image.open(BytesIO(response.content)).convert('L')
  pixels = list(img.getdata())
  w, h = img.size
  pixels = [pixels[i * w:(i + 1) * w] for i in range(h)]
  data_set.append(pixels)

data_set = np.array(data_set)
dense_data = data_set.reshape((2, 28 * 28))
dense_data = dense_data.astype('float32') / 255
conv_data = data_set.reshape((2, 28, 28, 1))
conv_data = conv_data.astype('float32') / 255

dense_pred = dense_model.predict_classes(dense_data)
conv_pred = conv_model.predict_classes(conv_data)

for i in range(len(data_set)):
  lbl = 'Dense: %s\nConv: %s\nTarget: %s' % (dense_pred[i], conv_pred[i], target_set[i])
  plt.figure(i)
  plt.xticks([])
  plt.yticks([])
  plt.title(lbl, fontsize=14)
  plt.imshow(data_set[i], cmap='gray')

plt.show()
